Project Overview

Crypto Watcher is a student web project created to practice server-side development using Node.js and Express.js. It demonstrates routing, request handling, form processing, and database interaction.

What We Added (MongoDB)

  • CRUD API for crypto assets: /api/assets
  • Filtering, sorting, and projection support
